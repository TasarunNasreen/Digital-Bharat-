# Digital Bharat

## Project Overview

Digital Bharat is an AI-powered civic companion for government services. This foundation uses Next.js, TypeScript, Tailwind CSS, shadcn/ui conventions, local JSON mock data, and browser storage for client-side persistence.

## Problem Statement

Citizens often need to navigate multiple portals, documents, schemes, and complaint systems. Digital Bharat aims to simplify civic access through a unified, responsive interface with mock AI-guided next steps.

## Features

- Landing page with hero, features, statistics, why section, CTA, and footer
- Government portal style UI with white, blue, and light gray styling
- Responsive cards for desktop, tablet, and mobile
- Local JSON mock data for services, complaints, documents, schemes, notifications, and users
- Reusable layout, navigation, footer, and shadcn-style UI components
- Local storage and theme hooks for future client-side workflows

## Tech Stack

- Next.js 15
- TypeScript
- Tailwind CSS
- shadcn/ui conventions
- Local JSON data
- Local Storage


## Deployment (Vercel)

1. Push the repository to a Git provider.
2. Import the project in Vercel.
3. Use the default Next.js framework settings.
4. Deploy.
